# BeachLab

## Beach Volleyball Intelligence

BeachLab es una aplicación de escritorio para el análisis de torneos de Beach Volley.

Su objetivo es importar automáticamente información de torneos, almacenar los datos en una base PostgreSQL y generar análisis técnicos, estadísticos y reportes para entrenadores.

## Objetivos de la versión 1.0

- Importar torneos desde Volleyball World.
- Almacenar la información en PostgreSQL.
- Analizar participantes y resultados.
- Exportar informes a Excel.

## Tecnologías

- C#
- .NET 8 LTS
- Windows Forms
- PostgreSQL
- ClosedXML
